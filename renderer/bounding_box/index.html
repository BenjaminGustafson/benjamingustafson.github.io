Bounding box

Given a triangle, find the smallest rectangle that the triangle fits in.

Inputs

  • A triangle with vertices (x0, y0), (x1,y1), (x2, y2).

Output:

  • A rectangle with top left corner (bb_min_x, bb_min_y) and bottom right corner (bb_max_x, bb_max_y).

Algorithm: 

for each x in [x0, x1, x2]:
    if x > bb_max_x then bb_max_x = x
    if x < bb_min_x then bb_min_x = x
for each y in [y0, y1, y2]:
    if x > bb_max_y then bb_max_y = y
    if x < bb_min_y then bb_min_y = y
