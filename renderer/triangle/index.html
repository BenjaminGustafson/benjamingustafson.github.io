Function: Triangle

Input:

  • Clip coords
  • Shader
  • Zbuffer

Output:

  • Draws the triangle to the image

Algorithm: 

                    
compute perspective vertices
for each pixel in bounding box:
    bc_screen = barycentric(triangle, pixel)
    bc_clip = bc_screen / pts.z
    compute frag_depth
    if pixel is not on screen: 
        skip to next pixel
    if fragment shader discards fragment
        continue loop
    set zbuffer of pixel
    set pixel in image
